## Usage guidelines

The `Code` component can be used to highlight small snippets of code within other text. They are often used to share code with other developers, in a distinct and easily copyable state.

There are three `sizes` for `Code`, but by default it will inherit the `font-size` of it's parent. If you wish to use the component as a standalone element, without surrounding text, then you must specify the `size` as one of the other available values: `small`, `medium` or `large`.

`Code` also has an `inverted` option for use with darker backgrounds.

## Best practices

<BestPractices>
  <BestPractices.Recommended>

    - use for inline code snippets
    - use for highlighting short strings within other text
    - use shorter content, with several words at most

  </BestPractices.Recommended>
  <BestPractices.Discouraged>

    - use for multi-line code snippets&mdash;use [Code Block](/components/code-block) instead
    - use for editing code

  </BestPractices.Discouraged>
</BestPractices>

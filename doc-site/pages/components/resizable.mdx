## Usage guidance

`Resizable` allows for multiple elements to be resized by the user. No additional wrappers are required - each child of `Resizable` will be just put in a separate panel. It is also possible to nest multiple `Resizable` components and create even complex layouts that way.

`Resizable` supports two orientations via `layout` prop: `row` and `column`.

By default, panels are split equally, but their initial proportions could be customized by providing an array of `defaultProportions`. Keep in mind, that number of items in that array _must_ be equal to the number of `Resizable` children.

After the user resizes panels to their liking, the easiest way to store new proportions is to persist them e.g. in `localStorage` by using `onResizeEnd` callback.

## Usage guidance

A `CodeBlock` component can be used to highlight a large block of code, including single lines, whilst maintaining formatting. They are often used share code with other developers, in a distinct and easily copyable state.

There are three (3) `sizes` for `CodeBlock`, but in most cases the default value should be used. `CodeBlock` also has an `inverted` option for use with darker backgrounds.

It should never be placed inline and instead the [Code](/components/code) component should be used.

## Best practices

<BestPractices>
  <BestPractices.Recommended>

    - use to display code snippets for copy or as read-only
    - use for single or multi-line code snippets

  </BestPractices.Recommended>
  <BestPractices.Discouraged>

    - use for inline code snippets&mdash;use [Code](/components/code) instead
    - use for editing code

  </BestPractices.Discouraged>
</BestPractices>

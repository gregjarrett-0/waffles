## Usage guidance

A `SkillTag` can be used in course/track description cards & pages to indicate the skill level of those courses / tracks. It should be used to provide more visual information about these things, through supporting content and not on its own.

There are three options for the `level`, indicating the required skill to represent. These are: `beginner`, `intermediate` and `advanced`. Terminology used to describe these difficulties should also follow the same options.

When positioned near the top of a page, or used with the intent of catching the eye, then `variant='primary'` should be used. In contrast, `variant='secondary'` should be used in cases where the `SkillTag` is secondary to the surrounding content and not a focus.

## Best practices

<BestPractices>
  <BestPractices.Recommended>

    - use as alongside other content
    - use the correct variant for the use case

  </BestPractices.Recommended>
  <BestPractices.Discouraged>

    - use alone, without other content
    - override the colors of the indicators

  </BestPractices.Discouraged>
</BestPractices>

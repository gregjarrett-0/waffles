## Usage guidance

Use this utility component in conjunction with [Input](/components/input), [Text Area](/components/text-area) or [Select](/components/select) to enhance them with: an accessible **label**, an optional **description**, and an **error message**. It is also possible to configure which **required indicator** is shown next to the label. All other attributes should be passed to the input directly.

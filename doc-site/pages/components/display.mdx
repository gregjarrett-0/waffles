## Usage guidance

Use `Display` sparingly to create impact when the main goal is visual storytelling, and drawing user attention with typography. It does not have any sematic meaning and shouldn't be used as yet another [Heading](/components/heading/).

In general the `Display` content should be _sentence cased_.

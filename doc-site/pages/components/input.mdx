## Usage guidance

Use [Form Field](/components/form-field) utility component to enhance this component with: an accessible **label**, an optional **description**, an **error message** or to show **required indicator**.

## Accessibility

When using this component without `FormField` be mindful to provide `aria-label` or at least meaningful `placeholder` text.

## Usage guidance

`Portal` provide a way to render `children` into a DOM node appended at the end of `document.body`, which exists outside of reguler DOM hierarchy of the parent component. Useful for preventing parent styles from clipping or hiding content, e.g. tooltips, dropdowns, and modals.

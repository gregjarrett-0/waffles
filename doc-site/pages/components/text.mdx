## Usage guidance

In general, when dealing with text, [Paragraph](/components/paragraph) component should be used. This `Text` component is meant to be used mainly as a building block for other components by taking advantage of the `as` prop, or as an inline text wrapper.

It has already basic typography-related styles applied, such as `font-family`, `font-size`, and `font-weight`.
